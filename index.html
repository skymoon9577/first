네일 수채화 갤러리

HTML/CSS로 구현된 가벼운 데모 페이지

완성 이미지

아래 이미지는 빨강색으로 통일된 수채화 스타일 네일 아트 예시입니다.

이미지 비율을 유지하며 화면 크기에 맞춰 자동으로 축소/확대됩니다.

이 페이지의 구성

1) HTML

이 파일은 index.html 이며, 반응형 이미지와 간단한 카드 레이아웃을 포함합니다.

2) Python + Markdown

01_image_and_md.py 실행 시, 첨부 이미지를 설명하는 image_explanation.md를 생성합니다.

3) 웹 크롤링 예제

02_web_crawl_example.py는 초간단 크롤러 예제로, 특정 URL의 제목/링크를 추출해 crawl_result.md로 저장합니다.

초보자 전용: 아주 간단한 순서 가이드

  1. 프로젝트 폴더를 열고 index.html을 더블클릭하여 브라우저에서 확인합니다.
  2. 터미널에서 python 01_image_and_md.py를 실행해 설명용 마크다운을 만듭니다.
  3. python 02_web_crawl_example.py --url "https://example.com" 형태로 크롤링 예제를 실행합니다.

문제가 생기면 어느 단계까지 했는지 숫자로 알려주세요. (예: “2번까지 완료”)

개발자도구(F12)로 공통 태그 규칙 찾기

크롬 개발자도구의 Elements 탭에서 좌측 상단의 선택 도구(화살표 아이콘)를 누르면 화면의 특정 영역을 선택할 수 있습니다. 비슷한 아이템을 여러 개 선택해 공통 클래스/태그/속성을 관찰하고 그 패턴을 CSS 선택자로 정리하세요.

  • 예) 상품 카드가 모두 <div class="product-card"> 구조라면, CSS/크롤러에서 .product-card를 공통 선택자로 사용
  • 예) 제목이 항상 <h2 class="title"> 라면, 크롤러에서 select('h2.title')를 사용할 수 있습니다.
